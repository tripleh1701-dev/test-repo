### Prerequisites

- Git installed on your system
- Basic understanding of version control

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/systiva/test-repo.git
   ```

2. Navigate to the project directory:
   ```bash
   cd test-repo
   ```
